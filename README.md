## Running the project

First, run the development server via:

```bash
git clone https://github.com/UoaUOACS/UOACS-Portal.git
code ./uoacs-portal
# then
pnpm i
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the development build.
